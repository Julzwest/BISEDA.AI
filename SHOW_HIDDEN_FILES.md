## .nojekyll is Hidden - Here's How to Show It

---

## Method 1: Show Hidden Files in Finder

**In Finder window:**
1. **Press:** `Cmd+Shift+.` (Command + Shift + Period)
2. **Hidden files will appear** (they'll look faded/grayed out)
3. **You'll see:** `.nojekyll` appear
4. **Drag it** to GitHub

**To hide them again:** Press `Cmd+Shift+.` again

---

## Method 2: Create .nojekyll on GitHub (Easier!)

**If you can't find it, create it on GitHub:**

1. **Go to:** https://github.com/Julzwest/BISEDA-AI/tree/gh-pages
2. **Click:** "Add file" ‚Üí "Create new file"
3. **File name:** `.nojekyll` (with the dot at the start)
4. **Leave content EMPTY** (or just add `#` as a comment)
5. **Commit message:** `Add .nojekyll`
6. **Select:** "Commit directly to the `gh-pages` branch"
7. **Click:** "Commit new file"

---

## Upload These 3 Items First:

1. `index.html`
2. `404.html`
3. `assets/` folder

**Then create `.nojekyll` on GitHub** (Method 2 above)

---
