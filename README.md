# Biseda.ai

A mobile-first React app for improving dating and communication skills with AI assistance.

## Features

- **Chat Practice**: Voice AI conversation simulator with real-time feedback
- **Style Advisor**: Fashion advice for different occasions
- **Tips & Advice**: Comprehensive dating tips and conversation analysis

## Setup

1. Install dependencies:
```bash
npm install
```

2. Run development server:
```bash
npm run dev
```

The app runs standalone with mock AI responses. No external API configuration needed!

## Project Structure

- `src/pages/` - Main page components
- `src/components/` - Reusable UI components
- `src/api/` - API client configuration
- `functions/` - Backend functions (Base44)
- `entities/` - Data models

## Tech Stack

- React 18
- React Router
- Tailwind CSS
- Lucide Icons
- Browser Web Speech API (for text-to-speech)
- Mock AI responses (standalone, no external APIs)
