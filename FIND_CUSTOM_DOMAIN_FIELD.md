## Where to Enter Custom Domain

### Step 1: Locate Custom Domain Section

On the GitHub Pages settings page:

1. **Scroll down** to find the "Custom domain" section
2. **Look for:** A section titled "Custom domain" with an input field
3. **The input field might be:**
   - Empty (blank)
   - Or have placeholder text

### Step 2: Enter Domain

1. **Click in the input field** (it might be empty or have placeholder text)
2. **Type:** `bisedaai.com`
3. **Click:** "Save" button (below the input field)

---

## If You Don't See Custom Domain Section

If you can't find it:

1. **Make sure you're on:** https://github.com/Julzwest/BISEDA.AI/settings/pages
2. **Scroll down** - it should be below "Build and deployment" section
3. **Look for:** Section titled "Custom domain"
